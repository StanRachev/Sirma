# Custom List

## Explanation
    Generic data structure that store any type, which are comparable. Test with String.

## Main class
- Console application that reads a command:
    - "**Add {element}**" - adds the element to the end of the list
    - "**Remove {index}**" - removes the element at the index
    - "**Contains {element}**" - prints true if contains and false if otherwise
    - "**Swap {index1} {index2}**" - swaps the elements at the given indexes
    - "**Greater {element}**" - counts the elements that are greater than the element
    - "**Max**" - prints the maximum element
    - "**Min**" - prints the minimum elements
    - "**Print**" - prints all elements
    - "**end**" - stops the program
