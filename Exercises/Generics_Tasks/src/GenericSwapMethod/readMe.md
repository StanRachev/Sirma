# Generic Swap

## Explanation
    Class Swap that swaps two indices of a list. Test with String and Integer.

## Main class
- Console application that reads **n** on the first line. On the next **n** lines are added the elements. On the last line are added two values, separated with space.
    
### Input:
- 3
- Rado
- Stan
- Val
- 1 2

### Output:
- java.lang.String: Rado
- java.lang.String: Val
- java.lang.String: Stan
